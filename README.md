### 프로젝트 실행

```
  yarn dev
```

### 요구사항

- [x] 올클리어 구현
- [x] 나누기 구현
- [x] 더하기 구현
- [x] 빼기 구현
- [x] 곱하기 구현
- [x] 마지막으로 "=" 클릭시 연산 완료
- [x] 이전 값을 베이스로 연산한다.
- [x] 키보드 입력이 아닌 클릭을 통해 숫자가 입력된다.
